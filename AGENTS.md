# SYNAPSE Lite — Runbook (2-Agent)

## Workflow

1. **Write `session/problem.md`** with the problem, context, and success criteria
2. **Spawn @explore** — code-level analysis
3. **Spawn @general** — architecture & design
4. **Read both opinions** from `session/opinions/`
5. **Decide yourself** — no consensus engine in Lite
6. **Implement** based on the stronger analysis

## Rules

- ZERO paid APIs
- File bus: all communication through markdown files
