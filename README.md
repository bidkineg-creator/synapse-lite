# SYNAPSE Lite

**Free 2-agent AI debate workflow for OpenCode.** No API keys. No subscriptions. Zero cost.

## What is this?

SYNAPSE Lite runs a simplified debate between 2 AI agents (@explore + @general) to analyze your code problems. They think independently, then you read their opinions and decide.

## What you get

- `AGENTS.md` — the runbook (2-agent version)
- `prompts/explore.md` — code-level analysis agent
- `prompts/general.md` — architecture & design agent
- `QUICKSTART.md` — run in 3 minutes

## Usage

```bash
opencode
# Tell the agent: "Read AGENTS.md and run a SYNAPSE session on: [your problem]"
```
