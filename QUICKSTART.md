# SYNAPSE Lite — Quickstart

## 1. Install OpenCode
```bash
npm install -g opencode-ai
```

## 2. Open this folder
```bash
cd synapse-lite
opencode
```

## 3. Run a session
Tell the agent:
```
Read AGENTS.md and run a SYNAPSE session on: [describe your code problem]
```

## 4. Read the results
Check `session/opinions/` for the agents' analysis.
